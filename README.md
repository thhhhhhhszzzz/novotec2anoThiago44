**1. Metadados:**
   ```html
   <!DOCTYPE html>
   <html lang="en">
   <head>
       <meta charset="UTF-8">
       <meta name="viewport" content="width=device-width, initial-scale=1.0">
       <title>Página de Exemplo</title>
   </head>
   <body>
       <!-- Conteúdo da Página -->
   </body>
   </html>
   ```

**2. Metadados (segundo exemplo):**
   ```html
   <!DOCTYPE html>
   <html lang="pt-br">
   <head>
       <meta charset="UTF-8">
       <meta name="description" content="Exemplo de utilização de metadados em HTML">
       <meta name="keywords" content="HTML, metadados, exemplo">
       <meta name="author" content="Seu Nome">
       <title>Metadados em HTML</title>
   </head>
   <body>
       <!-- Conteúdo da Página -->
   </body>
   </html>
   ```

**3. Separação de Conteúdo:**
   ```html
   <!DOCTYPE html>
   <html lang="en">
   <head>
       <meta charset="UTF-8">
       <meta name="viewport" content="width=device-width, initial-scale=1.0">
       <title>Página com Separação de Conteúdo</title>
   </head>
   <body>
       <header>
           <h1>Título da Página</h1>
           <nav>
               <!-- Links de Navegação -->
           </nav>
       </header>
       <main>
           <!-- Conteúdo Principal -->
       </main>
       <footer>
           <!-- Rodapé da Página -->
       </footer>
   </body>
   </html>
   ```

**4. Separação de Conteúdo (segundo exemplo):**

**7. Imagem:**
